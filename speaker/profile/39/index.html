Adel Daouzli

Version Française

A propos de moi

Docteur en sciences neuromorphiques, je suis actuellement ingénieur en systèmes embarqués, je suis co-fondateur du hackerspace Lyonnais le LOL (Laboratoire Ouvert Lyonnais), j'utilise Python depuis près de 10 ans pour différents usages.

Mon travail avec Python

Professionnellement:

  • J'ai développé un outil de validation de compilateurs.
  • J'ai développé un ensemble de tests utilisant py.test pour tester un système embarqué BSP Linux et une suite d'applications fonctionnant dessus.

Contributions au libre:

  • Je contribue quand j'ai du temps aux outils de tests en Python de Mozilla.
  • Je développe Pyment pour créer et convertir des docstrings.
  • Je développe le LOLOpener un système informant sur l'ouverture du LOL.
  • J'ai développé un petit serveur codelauncher permettant d'exécuter rapidement du code C ou Python dans un navigateur web.

Projets persos:

  • Je me suis développé une sorte de babyphone avec un côté serveur Python qui gère l'audio et l'envois sur un serveur Internet, et de l'autre côté une application Android pour être informé que bébé pleur.
  • J'ai développé différents outils et serveur en Python sur Raspberry Pi servers (Django, Flask, WSGI,...) pour gérer ma collection d'ebooks (ebooks), pour jouer de la musique après récupération sur Internet, gérer des vidéos en streaming en local,...

English version

About me

PhD in neuromorphic sciences, currently embedded software engineer, co-founder of the Lyon's hackerspace LOL (LyonOpenLab), I use Python for about 10 years and concerning several kinds of uses.

My Python work

Professional work:

  • I developed a specific compiler validating tool.
  • I developed a set of advanced tests using py.test to validate an embedded BSP and software applications.

Open Source contributions:

  • I contribute when I have time to Mozilla Python testing tools.
  • I'm developing Pyment for creating and converting docstrings.
  • I'm working on the LOLOpener a system to inform about the LOL opening status.
  • I developed a small server codelauncher to launch quickly some C or Python code in a web browser.

Personnal:

  • I developed a kind of babyphone: on one side a Python server manages the audio and sending on an Internet server, and on the other side a smartphone client (under Android) that notify when baby was crying.
  • I developed on Raspberry Pi servers in Python (Django, Flask, WSGI,...) to manage my ebook collection, playing music after getting it on Internet, managing streming local videos,...
